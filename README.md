# 로그인/회원가입 구현

## 환경
- node.js (10.16.0)
- express (4.16.1)
- webpack (4.39.3)

## 설명
- 네이버의 로그인/회원가입을 그대로 구현하는 토이 프로젝트
- 프론트엔드
  - js 및 정규표현식을 통한 input validation
  - modal
  - event처리
- 백엔드
  - 쿠키를 이용한 세션 구현
  - pug 사용
  - crypto 모듈을 이용한 패스워드 암호화 저장
  - 객체를 이용한 휘발성 메모리DB
- 백엔드 개발 기간에 Commit convention을 지키려 노력

## 구현기간
- 2019.08.28 ~ 2019.09.10
